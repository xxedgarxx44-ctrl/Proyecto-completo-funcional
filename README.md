# Proyecto-completo-funcional

"El proyecto consiste en una Plataforma Web Educativa e Interactiva llamada 'Pequeños Ingenieros'. Está diseñada específicamente para niños, con el objetivo de enseñar conceptos básicos de ingeniería (Civil, Mecatrónica y Estructuras) mediante la gamificación. La interfaz utiliza colores llamativos, tipografías claras y un sistema de recompensas visuales (barra de progreso) para mantener la motivación del alumno."
